# Stock Market News Sentiment Analysis & Summarization (LLM Project)

### Business Goal
Identify market sentiment trends from financial news articles and summarize them automatically to support better investment insights and faster signal interpretation.

### Tools & Methods
Python · Pandas · NumPy · Transformers · HuggingFace Models · NLP · Text Preprocessing · Prompt Engineering · EDA

### Approach
- Collected financial news text data
- Preprocessed and cleaned text
- Used transformer-based LLM for sentiment classification and summarization
- Analyzed weekly sentiment patterns vs market direction

### Result
LLM-based pipeline automated sentiment scoring and summarization, improving signal interpretation speed and enabling more data-driven trading insights.

### Next Steps / Improvement Ideas
- add Power BI dashboard for sentiment trend visualization
- test multiple base transformer architectures
- integrate real-time news source streaming API
